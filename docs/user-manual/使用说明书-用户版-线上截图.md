## 6. 推荐的上手顺序（给新用户）

1. 先看：总览首页（知道今天要干什么）
2. 再做：CRM 和客户中心（把客户信息沉淀好）
3. 每天用：消息中心（处理沟通，持续跟进）
4. 周复盘：订单 + 财务 + 库存（看交付与回款）
5. 管理员：设置中心 + 管理后台（管账号、权限、配置）

## 7. 常见问题

### Q1：我不是技术人员，能用吗？
可以。建议从 CRM、消息中心开始，用 2-3 天就能形成习惯。

### Q2：最容易出问题的是哪里？
不是系统本身，而是“跟进后不记录”。建议每次沟通后补一条记录。

### Q3：怎么判断团队有没有用好系统？
看三件事：
- 客户信息是否完整
- 沟通记录是否连续
- 重点客户是否按阶段推进
